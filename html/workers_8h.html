Detailed Description

Header file for worker functions and structures.

This header file declares structures and functions related to processing data chunks in parallel computing environments. It includes a structure for storing results of chunk processing and functions for processing chunks, checking if a character is a consonant, and checking if a word has two equal consonants. These utilities are essential for text analysis tasks in parallel processing applications.
