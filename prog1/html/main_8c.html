Detailed Description

Main program for parallel file processing using MPI.

This file contains the main logic for a parallel file processing application using MPI. It includes the initialization of MPI, setting up file structures, dispatching tasks to worker processes, and aggregating results. The program processes files in chunks, counting words and words with consonants.
